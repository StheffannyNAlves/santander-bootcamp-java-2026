# Santander Bootcamp 2026 - Java AI Back-end

Repositório central para o desenvolvimento e documentação do treinamento intensivo de back-end Java em parceria com a DIO e o Santander.

## 🎯 Objetivo
Registrar a evolução técnica, desde os fundamentos do Java e Orientação a Objetos até a construção de APIs robustas utilizando Spring Boot e integração com IA.

## 🛠️ Stack Tecnológica
* **Linguagem:** Java 21 (LTS)
* **Framework:** Spring Boot
* **Gerenciador de Dependências:** Maven
* **Ambiente:** Ubuntu / IntelliJ IDEA
